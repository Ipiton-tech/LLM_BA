# Voice2Doc

<small>_**Статус:** <span style="color: #54BA7E; font-weight: 600;">✓ Используем в работе</span>_</small>

Сервис предназначен для транскрибации аудиозаписей встреч и их последующей трансформации в документы по заданным шаблонам. Персональные данные анонимизируются перед отправкой во внешние системы.

## Доступ

> **URL:** [https://voice2doc.ipiton.ru/](https://voice2doc.ipiton.ru/)
> 
> **Авторизация:** Через учетную запись пользователя в компании (`Ivanov_ai`, пароль)

  *   Обработка интервью с пользователями.

## Механика работы с инструментом

1. **Загрузите аудио.**
  Поддерживаются все популярные форматы: `mp3`, `wav`, `m4a`, `mp4`, `webm`, `mkv`, `docx`, `pdf`.
  <br>
2. **Выберите шаблон.**
  Готовый из библиотеки или создайте свой.
  <br>
3. **Получите документ.**
  Скачайте результат и используйте его в дальнейшей работе.

<br>

{% note warning "! Конфиденциальность" %}

При использовании инструмента записи встреч, особенно с внешними участниками, **всегда предупреждайте их о записи** и целях ее использования.

В случае если предварительно запись не была включена, рекомендуется в ходе встречи предупредить участников о старте записи и **повторно проговорить** о договоренностях, утвержденных ранее.

{% endnote %}
