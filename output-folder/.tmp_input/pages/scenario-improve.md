## Шаг 1: Генерация уточняющих вопросов

В рамках задачи по подготовке ТЗ на доработку функциональности инструмента, ИИ может выступать в роли ассистента при написании ТЗ, генерируя уточняющие вопросы, которые помогают выявить неполноту требований.

{% note tip %}

### Промт для генерации вопросов

Нужно доработать эту обработку. Необходимо добавить возможность по сохраненным настройкам производить автоматическую загрузку файлов с заранее заданной папки. После загрузки файлы должны удаляться. Составь ТЗ для разработчика. Во вложении формат, которому должно соответствовать ТЗ. Задай мне уточняющие вопросы, если это нужно.
