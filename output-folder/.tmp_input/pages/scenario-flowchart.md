# Работа с блок-схемами

<small>_**Статус:** <span style="color: #E74C3C; font-weight: 600;">✗ Не используем в работе</span> (15.04.2026)_</small>

## Возможности

ИИ успешно справляется с созданием простых блок-схем по текстовому описанию, а также с расшифровкой уже существующих блок-схем.

## Ограничения

Функционал по генерации и анализу блок-схем работает корректно **только для простых бизнес-процессов**. Для сложных процессов результат может быть некорректным.
