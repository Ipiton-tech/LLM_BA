# LLM в работе бизнес-аналитика

## Что такое LLM

LLM (Large Language Model) — это большая языковая модель (далее **«ИИ»**, искусственный интеллект); продвинутая программа, взаимодействующая с человеком на его языке.

### Ключевые принципы работы с ИИ

{% note info "" %}

*   При вдумчивом и умеренном использовании, ИИ может повысить качество задач, экономить время и развивать навыки специалиста.
*   **Результаты, сгенерированные ИИ, требуют обязательной проверки и доработки человеком.**
    **ИИ — это помощник, а не замена специалиста.**

{% endnote %}
