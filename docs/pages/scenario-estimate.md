# Оценка трудоемкости задач

<small>_**Статус:** <span style="color: #E74C3C; font-weight: 600;">✗ Не используем в работе</span> (15.04.2026)_</small>

## Почему не используем

Использование ИИ для оценки задач в человеко-часах — сомнительно:

*   **Нехватка контекста.** Из-за отсутствия информации о состоянии кода, техническом долге и готовности связанных задач, ИИ дает неточную оценку.
*   **Требуется глубокая экспертиза.** Использование ИИ для таких задач требует от аналитика умения корректно интерпретировать результаты и отсеивать нерелевантные выводы.
