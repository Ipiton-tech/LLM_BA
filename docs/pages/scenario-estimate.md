# Оценка трудоемкости задач

<small>_**Статус:** <span style="color: #E74C3C; font-weight: 600;">✗ Не используем в работе</span> (15.04.2026)_</small>

## Почему не используем

Использование ИИ для оценки задач в человеко-часах — сомнительно:

<div style="margin-left: 20px;">
