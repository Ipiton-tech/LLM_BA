# Работа с блок-схемами

<small>_**Статус:** <span style="color: #E74C3C; font-weight: 600;">✗ Не используем в работе</span> (15.04.2026)_</small>

## <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" width="24" height="24"><title>chevron-double-right</title><path d="M5.59,7.41L7,6L13,12L7,18L5.59,16.59L10.17,12L5.59,7.41M11.59,7.41L13,6L19,12L13,18L11.59,16.59L16.17,12L11.59,7.41Z" /></svg>   Возможности

ИИ успешно справляется с созданием простых блок-схем по текстовому описанию, а также с расшифровкой уже существующих блок-схем.

## <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" width="24" height="24"><title>chevron-double-right</title><path d="M5.59,7.41L7,6L13,12L7,18L5.59,16.59L10.17,12L5.59,7.41M11.59,7.41L13,6L19,12L13,18L11.59,16.59L16.17,12L11.59,7.41Z" /></svg>   Ограничения

Функционал по генерации и анализу блок-схем работает корректно **только для простых бизнес-процессов**. Для сложных процессов результат может быть некорректным.
